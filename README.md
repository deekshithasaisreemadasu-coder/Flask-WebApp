# Flask-WebApp

## Description
This Flask web application allows users to register, login, and submit feedback. It demonstrates backend routing, form handling, and template rendering using Python and Flask.

## Features
- User registration and login
- Feedback form submission
- Template rendering with HTML/CSS
- Easy setup and run locally

## Tools & Technologies
- Python
- Flask
- HTML / CSS
- Jinja2 templates
